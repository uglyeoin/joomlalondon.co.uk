Download Shadowbox Media Viewer version 4.0 for Joomla 1.5.x or version 4.1 for Joomla 2.5.x and 3.x

Shadowbox is an online media viewer application that supports all of the web's most popular media publishing formats. Shadowbox is written entirely in JavaScript and CSS and is highly customisable. Using Shadowbox, website authors can showcase a wide assortment of media in all major browsers without navigating users away from the linking page.

View Shadowbox Media Viewer Demo

Although Shadowbox can be used in standalone mode, it comes bundled with adapters for the following JavaScript frameworks:

  • Prototype
  • jQuery
  • MooTools (requires 1.2 Core)
  • Dojo Toolkit
  • Yahoo! User Interface Library (requires yahoo-dom-event.js)
  • Ext (requires ext-core.js)
For easy thumbnail creation, try mavik Thumbnails set to Slimbox and no link to scripts

Installation Instructions
  1. Install/Upgrade the plugin through the normal Joomla Install/Uninstall page
  2. Open Shadowbox Media Viewer in the Plugin Manager
  3. Customise the parameters below and publish the plugin
If upgrading, you do not have to uninstall first but you may have to reselect your language

Usage Instructions
The simplest way to tell Shadowbox which links you want it to open is to add a rel="shadowbox" attribute to each link. For example, say you have this link to an image on your page:

My ImageIn order to set up this link for use with Shadowbox, simply change it to this:

My ImageThat's it! Clicking on this link should now open up the image in Shadowbox.

The word lightbox will also work here. This feature was added for compatibility with the original Lightbox script. Also note that because HTML area tags do not support the rel attribute, you cannot use this method to set them up for use with Shadowbox.

If you would like to display a title for your image, simply add a title attribute to the link.

My ImageIn addition to displaying single images and movies, Shadowbox is also capable of displaying galleries of content. In order to designate a link as part of a gallery, you must add the gallery name to the rel attribute between square brackets immediately following the word shadowbox.
